File Inclusion

Include 함수를 이용해 include 할 파일을 웹 요청을 통해 지정할 수 있는 경우, 값을 조작하여 원하는 파일을 처리

If a file to be included can be specified through a web request using the Include function, the desired file is processed by manipulating the value.
