WEB_HACKING

Web Hacking is also called web application hacking, and it uses all security that can occur on web services to perform malicious actions. The attack mainly targets web services, but it can also attack frameworks(Spring) and web application servers.

웹 어플리케이션 해킹 이라고도 부르고 웹 서비스 상에서 발생될 수 있는 모든 보안 허점(Security Hole)을 이용해 악의적인 행위를 하는것이다. 웹 서비스를 대상으로 공격을 주로하지만 프레임워크(Spring), 웹 어플리케이션 서버 대상으로 공격을 수행할수도 있다.
