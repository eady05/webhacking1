File Upload

웹 애플리케이션의 파일 업로드 할 수 있는 기능을 악용하여 악의적인 파일(웹쉘) 업로드를 통해 시스템 권한을 장악하는 기법을 의미함

This refers to a technique to take over system privileges through malicious file (webshell) uploads by exploiting the file upload function of the web application.
