CSRF

사이트 간 요청 위조는 웹사이트 취약점 공격의 하나로, 사용자가 자신의 의지와는 무관하게 공격자가 의도한 행위(수정, 삭제, 등록 등)를 특정 웹사이트에 요청하게 하는 공격

Cross-site request forgery is one of the website vulnerability attacks, in which the user makes a request to a specific website for the intended action (modification, deletion, registration, etc.) of the attacker regardless of his/her will.
